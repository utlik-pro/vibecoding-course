Итоговый проект: SaaS Dashboard

Финальное задание

Задание

Создайте полноценный SaaS Dashboard с использованием всех изученных инструментов Claude Code.

Важно

Это финальный проект курса. Объедините Landing Page из модуля 11 и Dashboard из модуля 12 в единый продукт.

Требования

Dashboard (обязательно):

  • Минимум 4 страницы (Overview, Analytics, Reports, Settings)
  • Интерактивные графики (Recharts)
  • Таблицы с данными
  • Формы с валидацией

Интеграции (обязательно):

  • Подключен GitHub MCP
  • Минимум 3 Custom Skills
  • Настроены Hooks

Деплой (обязательно):

  • Репозиторий на GitHub
  • Деплой на Vercel
  • Рабочий URL
