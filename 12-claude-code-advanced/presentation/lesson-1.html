MCP серверы - суперсилы Claude

45 минут

Что такое MCP?

MCP (Model Context Protocol) — протокол для подключения внешних инструментов к Claude.

Аналогия

MCP серверы = Плагины для браузера

Claude + GitHub MCP = Claude может создавать PR напрямую
Claude + Figma MCP = Claude может читать дизайн

Без MCP С MCP
Читает/пишет файлы + работает с GitHub
Выполняет bash + читает Figma дизайн
Поиск по проекту + SQL запросы

Подключение MCP серверов

Используйте команду claude mcp add:

HTTP сервер (рекомендуется):

Terminal
$ claude mcp add --transport http github https://api.githubcopilot.com/mcp/

Stdio сервер (локальный):

Terminal
$ claude mcp add --transport stdio db -- npx -y @bytebase/dbhub --dsn "..."

Управление серверами:

  • claude mcp list — список серверов
  • claude mcp get github — детали сервера
  • claude mcp remove github — удаление
  • /mcp — статус в Claude Code

Популярные MCP серверы

GitHub

$ claude mcp add --transport http github https://api.githubcopilot.com/mcp/

Возможности: Issues, PR, коммиты, ветки, code review

Sentry (мониторинг ошибок)

$ claude mcp add --transport http sentry https://mcp.sentry.dev/mcp

Возможности: Ошибки, стектрейсы, дебаг production

PostgreSQL

$ claude mcp add --transport stdio db -- npx -y @bytebase/dbhub --dsn "postgresql://..."

Возможности: SQL на естественном языке, анализ схемы

Notion

$ claude mcp add --transport http notion https://mcp.notion.com/mcp

Возможности: Работа с документами, базами знаний

Практика: Подключаем GitHub MCP

Шаг 1: Добавляем GitHub MCP

Terminal
$ claude mcp add --transport http github https://api.githubcopilot.com/mcp/

Шаг 2: Запускаем Claude Code

Terminal
$ claude

Шаг 3: Аутентификация

Claude Code
> /mcp
# Выберите "Authenticate" для GitHub

Шаг 4: Тестируем

Claude Code
> Show me my recent repositories

> Create an issue in my-project: "Bug fix needed"
