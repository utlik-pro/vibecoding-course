Субагенты и Task Tool

45 минут

Что такое субагенты?

Субагенты — это специализированные "помощники" Claude.

Каждый субагент:

  • Имеет свой контекст (не видит весь разговор)
  • Специализируется на определённых задачах
  • Может работать параллельно

Типы субагентов:

ТипНазначениеИнструменты
ExploreИсследование кодбейзаRead, Glob, Grep
PlanПланирование реализацииRead, Glob, Grep
BashВыполнение командBash
general-purposeУниверсальные задачиВсе

Task Tool — оркестрация

Task Tool позволяет запускать субагентов.

Когда использовать:

  • Исследование большого кодбейза
  • Параллельное выполнение задач
  • Сложные многошаговые операции
Claude Code
> Исследуй как работает авторизация в проекте

# Claude запустит Explore агента
# Агент найдёт файлы, прочитает и вернёт summary

Параллельное выполнение

Можно запустить несколько агентов одновременно!

Claude Code
> Параллельно:
1. Исследуй структуру API
2. Найди все компоненты с формами
3. Проверь конфигурацию TypeScript

# 3 Explore агента работают одновременно!
Когда это полезно

Анализ большого проекта, подготовка к рефакторингу, аудит кодбейза.

Создание своих субагентов

Можно создать собственного субагента в .claude/agents/:

Пример: Security Reviewer

# .claude/agents/security-reviewer.md
---
name: security-reviewer
description: Reviews code for security vulnerabilities
tools: Read, Grep, Glob
model: opus
---

You are a security engineer. Review code for:
1. SQL injection
2. XSS vulnerabilities
3. Hardcoded secrets
4. OWASP Top 10

Provide specific line references.

Использование:

> Use security-reviewer to check the auth module
Frontmatter опции

name — имя агента
tools — доступные инструменты
model — модель (sonnet, opus, haiku)

Практика

Задание 1: Исследование

> Используй Explore агента чтобы понять структуру проекта

Задание 2: Свой агент

> Создай .claude/agents/code-reviewer.md
для проверки качества кода

Задание 3: Параллельные задачи

> Параллельно выполни:
1. Найди все TODO комментарии
2. Найди все console.log
3. Проверь безопасность с security-reviewer
