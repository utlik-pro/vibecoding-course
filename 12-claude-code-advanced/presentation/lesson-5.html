Финальный спринт - Dashboard

45 минут

План Dashboard

Dashboard Layout
├── Sidebar (навигация)
├── Header (профиль, уведомления)
└── Main Content
    ├── Overview (метрики)
    ├── Analytics (графики)
    ├── Reports (таблицы)
    └── Settings (настройки)

Основной промпт

Создай Dashboard для SaaS Analytics:

## Layout
- Sidebar слева с навигацией
- Header сверху с профилем
- Main content area

## Страницы
1. Overview — 4 карточки метрик + график
2. Analytics — интерактивные графики
3. Reports — таблица с данными
4. Settings — форма настроек

## Компоненты
- Карточка метрики (иконка, значение, изменение)
- Линейный график (Recharts)
- Круговая диаграмма
- Таблица с сортировкой
- Форма с валидацией

## Дизайн
- Цвета как на Landing
- Тёмный sidebar
- Светлый main content

Используй Next.js 14, TypeScript, Tailwind, Recharts.

Интеграция инструментов

Используем MCP для GitHub:

> Создай репозиторий saas-dashboard на GitHub
> Сделай initial commit

Используем Skills:

> /test
> /deploy

Используем субагенты:

> Параллельно проверь:
1. Все ли страницы рендерятся
2. Есть ли TypeScript ошибки
3. Корректен ли адаптив

Деплой

> Задеплой проект на Vercel и верни URL

# Claude:
# 1. Проверит build
# 2. Подключит репозиторий
# 3. Запустит деплой
# 4. Вернёт URL
