Итоговый проект: SaaS Landing Page

Самостоятельная работа

Задание

Создайте landing page для SaaS Analytics Dashboard, используя Claude Code.

Важно

Весь код должен быть создан через Claude Code, не вручную. Это главное условие проекта!

Требования

Обязательные секции:

  • Hero секция — заголовок, подзаголовок, CTA кнопка
  • Features — минимум 3 карточки с преимуществами
  • Pricing — минимум 2 тарифа (Free и Pro)
  • Testimonials — минимум 2 отзыва
  • Footer — навигация и копирайт

Технические требования:

  • Next.js 14 или обычный HTML/CSS
  • Tailwind CSS для стилей
  • Адаптивный дизайн (mobile-first)
  • Код на GitHub

Файлы проекта:

  • CLAUDE.md — описание проекта для Claude
  • .claude/settings.json — настройки разрешений
  • Минимум 3 git коммита

Бонусные задания

За дополнительные баллы:

  • +10: Анимации при скролле
  • +10: Тёмная тема
  • +10: Форма подписки с валидацией
  • +10: Деплой на Vercel

Подсказки

Начните с CLAUDE.md

Первым делом создайте CLAUDE.md с описанием проекта. Это поможет Claude лучше понимать контекст.

Итерируйте

Не пытайтесь создать идеальную страницу с первого промпта. Создайте базу, затем улучшайте по частям.

Коммитьте часто

Делайте коммит после каждой законченной секции. Это и для git истории, и для безопасности.
