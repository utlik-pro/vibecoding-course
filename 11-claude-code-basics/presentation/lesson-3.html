Инструменты для работы с файлами

Claude Code использует специальные инструменты (tools) для работы:

Инструмент Что делает Пример промпта
Read Читает файл "Покажи app/page.tsx"
Write Создаёт файл "Создай utils/helpers.ts"
Edit Редактирует файл "Добавь кнопку в header"
Glob Ищет файлы "Найди все .tsx файлы"
Grep Ищет текст "Где используется useState"
Как это работает

Когда вы говорите: "Добавь кнопку логина в header"

Claude автоматически:

  1. Read - читает текущий header
  2. Понимает структуру
  3. Edit - добавляет кнопку в нужное место

Чтение и анализ кода

Примеры промптов для анализа:

Claude Code
> Покажи структуру проекта

> Объясни что делает app/api/users/route.ts

> Найди все компоненты с useState

> Как связаны файлы в папке components/?
Claude сам решает что читать!

Когда вы спрашиваете "как работает авторизация", Claude сам найдёт нужные файлы, прочитает их и объяснит логику.

Создание и редактирование

Создание файлов

Claude Code
> Создай компонент Button с variants:
primary, secondary, danger

> Создай страницу /dashboard с layout

> Создай API route для списка пользователей

Редактирование файлов

Claude Code
> Добавь тёмную тему в этот компонент

> Исправь баг: кнопка не работает на мобильных

> Рефакторинг: вынеси логику в отдельный хук
Важно

Claude спросит разрешение перед изменением файлов! Вы всегда контролируете что меняется.

Поиск по проекту

Glob — поиск файлов по паттерну

Claude Code
> Найди все файлы .test.ts

> Покажи все компоненты в components/

Grep — поиск текста в файлах

Claude Code
> Найди все места где используется fetch

> Где в проекте определён тип User?

Plan Mode — планирование

Plan Mode позволяет сначала спланировать изменения.

Claude Code
# Нажмите Ctrl+G для входа в Plan Mode

> Добавь авторизацию через Google

# Claude создаст план, а не сразу код

Рекомендуемый workflow:

  1. Explore — изучение кодбейза
  2. Plan — планирование (Ctrl+G)
  3. Implement — реализация
  4. Commit — сохранение
Checkpoints

Claude автоматически сохраняет checkpoints. Используйте /rewind или Esc+Esc чтобы откатить изменения.
