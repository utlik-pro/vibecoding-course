Установка и первый проект

30 минут

Что такое Claude Code?

Claude Code — это AI-программист в вашем терминале.

В отличие от ChatGPT или Claude.ai в браузере, Claude Code работает напрямую с вашими файлами:

ChatGPT / Claude.ai Claude Code
Работает в браузере Работает в терминале
Копируете код вручную Пишет код прямо в файлы
Не видит ваш проект Видит весь проект целиком
Нужно объяснять контекст Сам понимает структуру
Где работает Claude Code

В терминале (основной режим), в браузере на claude.ai/code, в Desktop приложении, в VS Code и JetBrains IDE.

Установка Claude Code

Перед началом

  • Открытый терминал или командная строка
  • Проект с кодом для работы
  • Подписка Claude (Pro, Max, Teams, Enterprise) или API ключ
Где работает Claude Code

В терминале (CLI), в браузере (claude.ai/code), Desktop приложении, VS Code, JetBrains IDE, Slack, и в CI/CD через GitHub Actions и GitLab.

Выберите способ установки

macOS / Linux / WSL:

Terminal
$ curl -fsSL https://claude.ai/install.sh | bash

Windows PowerShell:

PowerShell
> irm https://claude.ai/install.ps1 | iex

Windows CMD:

CMD
> curl -fsSL https://claude.ai/install.cmd -o install.cmd && install.cmd
Автообновление

Native-установка автоматически обновляется в фоне.

Terminal
$ brew install --cask claude-code
Без автообновления

Homebrew не обновляет автоматически. Запускайте brew upgrade claude-code периодически.

PowerShell
> winget install Anthropic.ClaudeCode
Без автообновления

Обновляйте через winget upgrade Anthropic.ClaudeCode.

Первый запуск

Перейдите в каталог проекта
Terminal
$ cd /path/to/your/project
Запустите Claude Code
Terminal
$ claude
# При первом запуске — вход через браузер
Задайте первый вопрос
Claude Code
> what does this project do?
Подсказка

Claude Code читает файлы проекта по мере необходимости. Вам не нужно вручную добавлять контекст — просто задайте вопрос.

Команда Что делает

Практика: Первый проект за 15 минут

Задание

Создаём landing page для фитнес-приложения FitLife с помощью Claude Code.

Шаг 1: Создаём папку проекта

Terminal
$ mkdir fitness-landing && cd fitness-landing

Шаг 2: Запускаем Claude Code

Terminal
$ claude

Шаг 3: Даём промпт

Создай landing page для фитнес-приложения FitLife:
- Современный дизайн с градиентами
- Hero секция с заголовком и CTA
- 3 карточки с преимуществами
- Секция с отзывами
- Footer

Используй HTML, CSS (Tailwind через CDN),
и минимум JavaScript.

Шаг 4: Смотрим результат

Terminal
$ open index.html
# На Windows: start index.html
Результат

Через 2-3 минуты у вас готовый landing page! Claude сам создаст структуру файлов и напишет весь код.

Советы для начинающих

Вместо: "fix the bug"

Попробуйте: "fix the login bug where users see a blank screen after entering wrong credentials"

Разбивайте сложные задачи на этапы:

1. create a new database table for user profiles
2. create an API endpoint to get and update profiles
3. build a webpage for viewing and editing profiles

Перед внесением изменений дайте Claude понять код:

analyze the database schema
explain the folder structure

? — все сочетания клавиш, Tab — автодополнение, ↑ — история команд, / — все команды и skills.
