Что такое Claude Code?

Claude Code — это AI-программист в вашем терминале.

В отличие от ChatGPT или Claude.ai в браузере, Claude Code работает напрямую с вашими файлами:

ChatGPT / Claude.ai Claude Code
Работает в браузере Работает в терминале
Копируете код вручную Пишет код прямо в файлы
Не видит ваш проект Видит весь проект целиком
Нужно объяснять контекст Сам понимает структуру
Где работает Claude Code

В терминале (основной режим), в браузере на claude.ai/code, в Desktop приложении, в VS Code и JetBrains IDE.

Установка Claude Code

Требования

  • macOS, Linux или Windows
  • Подписка Claude Pro/Max/Teams ($20+/мес) или API ключ

Установка (выберите ваш способ)

macOS / Linux / WSL:

Terminal
$ curl -fsSL https://claude.ai/install.sh | bash

Windows PowerShell:

PowerShell
> irm https://claude.ai/install.ps1 | iex

Homebrew (macOS):

Terminal
$ brew install --cask claude-code

Проверка установки

Terminal
$ claude --version

Первый запуск

Terminal
$ cd your-project && claude
# Откроется браузер для авторизации
Подсказка

При первом запуске откроется браузер для авторизации. Поддерживаются: Claude Pro, Max, Teams, Enterprise и Console (API).

Базовые команды

Запуск Claude Code

Terminal
# Интерактивный режим
$ claude

# Одна команда без диалога
$ claude -p "создай файл hello.js"

Slash-команды внутри Claude

Команда Что делает
/help Показать все команды
/clear Очистить контекст (начать заново)
/init Создать CLAUDE.md для проекта
/login Сменить аккаунт
/compact Сжать контекст
/permissions Настроить разрешения

Навигация

Клавиши Действие
Enter Отправить сообщение
Shift+Enter Новая строка
Esc Остановить действие
Ctrl+G Plan Mode (только чтение)
? Показать все shortcuts
exit или Ctrl+C Выйти из Claude Code

Практика: Первый проект за 15 минут

Задание

Создаём landing page для фитнес-приложения FitLife с помощью Claude Code.

Шаг 1: Создаём папку проекта

Terminal
$ mkdir fitness-landing && cd fitness-landing

Шаг 2: Запускаем Claude Code

Terminal
$ claude

Шаг 3: Даём промпт

Создай landing page для фитнес-приложения FitLife:
- Современный дизайн с градиентами
- Hero секция с заголовком и CTA
- 3 карточки с преимуществами
- Секция с отзывами
- Footer

Используй HTML, CSS (Tailwind через CDN),
и минимум JavaScript.

Шаг 4: Смотрим результат

Terminal
$ open index.html
# На Windows: start index.html
Результат

Через 2-3 минуты у вас готовый landing page! Claude сам создаст структуру файлов и напишет весь код.
