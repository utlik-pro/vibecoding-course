Hero Section
├── Заголовок + подзаголовок
├── CTA кнопка

Features Section
├── Карточка 1: Аналитика в реальном времени
├── Карточка 2: Простые отчёты
└── Карточка 3: Интеграции

├── Free план (0₽)

├── Ссылки

Шаг 2: Создание Landing Page

Основной промпт для Claude

Создай landing page для SaaS Analytics Dashboard.

## Дизайн
- Современный, минималистичный
- Основной цвет: синий (#3B82F6)
- Акцент: зелёный (#10B981)
- Тёмный текст на светлом фоне

## Hero секция
Заголовок: "Аналитика, которая работает на вас"
Подзаголовок: "Отслеживайте метрики, создавайте
отчёты, принимайте решения на основе данных"
CTA: "Начать бесплатно" + "Посмотреть демо"

## Features (3 карточки)
1. Дашборд в реальном времени
   - Иконка: график
   - Описание: Следите за ключевыми метриками

2. Простые отчёты
   - Иконка: документ
   - Описание: Создавайте отчёты в один клик

3. Интеграции
   - Иконка: пазл
   - Описание: Подключите инструменты

## Pricing
- Free: 0₽/мес - базовая аналитика
- Pro: 1490₽/мес - расширенные функции
- Enterprise: по запросу - для больших команд

Используй Next.js 14, TypeScript, Tailwind CSS.
Совет

Чем детальнее промпт — тем лучше результат. Не стесняйтесь указывать цвета, тексты, структуру.
