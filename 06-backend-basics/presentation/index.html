Режим студента

Backend для вайбкодера

Создаём работающий бэкенд с помощью ИИ.
API, база данных, CRUD операции — всё с нуля.

Заметка для преподавателя

Общая продолжительность модуля: ~3 часа

Рекомендация: Этот модуль знакомит с бэкендом людей с нулевым опытом. Используйте аналогии (ресторан, Excel). Показывайте экран и делайте всё вместе со студентами.

Ключевая цель: К концу модуля у каждого студента будет работающее приложение Money Tracker с базой данных и API.

Что вы создадите в этом модуле:
  • Поймёте что такое бэкенд, API и база данных
  • Настроите Supabase и создадите таблицу
  • Напишете API endpoints с помощью Cursor
  • Реализуете CRUD операции (создание, чтение, обновление, удаление)
  • Соедините фронтенд с бэкендом
  • Создадите приложение Money Tracker
Проект модуля: Money Tracker
Приложение для учёта личных финансов — добавление доходов и расходов, просмотр баланса, удаление транзакций. Реальный проект с реальной базой данных!
