Фильтрация данных — WHERE

Выбираем только нужные записи из таблицы

Практика 1: Простая фильтрация

WHERE позволяет выбрать только записи, соответствующие условию:

SQL 
-- Только важные задачи
SELECT * FROM tasks WHERE priority = 'high';

-- Только невыполненные
SELECT * FROM tasks WHERE completed = false;
Запомните: Текстовые значения пишем в одинарных кавычках: 'high', а не "high"
Практика 2: Операторы сравнения

SQL поддерживает стандартные операторы сравнения:

Оператор Значение Пример
=Равноpriority = 'high'
!= или <>Не равноpriority != 'low'
>Большеid > 5
<Меньшеid < 3
>=Больше или равноid >= 2
<=Меньше или равноid <= 5
SQL 
-- Задачи с id больше 3
SELECT * FROM tasks WHERE id > 3;

-- Все кроме низкого приоритета
SELECT * FROM tasks WHERE priority != 'low';
Практика 3: Комбинируем условия (AND / OR)
SQL 
-- Важные И невыполненные (оба условия должны быть true)
SELECT * FROM tasks
WHERE priority = 'high' AND completed = false;

-- Важные ИЛИ средние (хотя бы одно условие true)
SELECT * FROM tasks
WHERE priority = 'high' OR priority = 'medium';

AND

ОБА условия должны быть true

high AND not completed

OR

ХОТЯ БЫ ОДНО условие true

high OR medium

Практика 4: Оператор IN

IN — удобная замена нескольким OR:

SQL 
-- Вместо: priority = 'high' OR priority = 'medium'
SELECT * FROM tasks
WHERE priority IN ('high', 'medium');

-- Задачи с определёнными id
SELECT * FROM tasks
WHERE id IN (1, 3, 5, 7);
Практика 5: Поиск по тексту (LIKE)

LIKE ищет по шаблону в текстовых полях:

SQL 
-- Задачи со словом "книг" в названии (в любом месте)
SELECT * FROM tasks WHERE title LIKE '%книг%';

-- Задачи начинающиеся на "По"
SELECT * FROM tasks WHERE title LIKE 'По%';

-- Задачи заканчивающиеся на "ку"
SELECT * FROM tasks WHERE title LIKE '%ку';
Шпаргалка:
  • % — любые символы (ноль или больше)
  • _ — ровно один символ
Практика 6: Работа с NULL

NULL — специальное значение "нет данных". Проверяется через IS NULL:

SQL 
-- Задачи БЕЗ описания
SELECT * FROM tasks WHERE description IS NULL;

-- Задачи С описанием
SELECT * FROM tasks WHERE description IS NOT NULL;
Важно: description = NULL НЕ РАБОТАЕТ! Используйте только IS NULL
Проверка знаний

Как найти задачи с высоким ИЛИ средним приоритетом?

WHERE priority = 'high' AND priority = 'medium'
WHERE priority IN ('high', 'medium')
WHERE priority = 'high, medium'
WHERE priority LIKE 'high%medium'
Мини-задания (делаем вместе)
  1. Найдите все задачи с низким приоритетом
  2. Найдите невыполненные задачи с описанием
  3. Найдите задачи где в названии есть буква "а"
Показать решения 
-- 1. Низкий приоритет
SELECT * FROM tasks WHERE priority = 'low';

-- 2. Невыполненные с описанием
SELECT * FROM tasks WHERE completed = false AND description IS NOT NULL;

-- 3. Буква "а" в названии
SELECT * FROM tasks WHERE title LIKE '%а%';
