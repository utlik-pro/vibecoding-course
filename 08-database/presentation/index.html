Заметка для преподавателя

Общая продолжительность модуля: ~3.5 часа

Формат: 90% практики. Каждый урок — работа в SQL Editor. Студенты копируют запросы и сразу видят результат.

Ключевая цель: К концу модуля студенты умеют писать SQL запросы с JOIN и GROUP BY, понимают связи между таблицами.

Что вы освоите в этом модуле:
  • SQL запросы: SELECT, INSERT, UPDATE, DELETE
  • Фильтрация и сортировка данных
  • Связи между таблицами (Foreign Key)
  • JOIN для объединения данных
  • Агрегация: COUNT, SUM, AVG, GROUP BY
  • Интеграция SQL в приложение
