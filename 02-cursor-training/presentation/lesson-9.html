Структура проекта

Организуем файлы правильно и связываем их между собой

Заметка для преподавателя

Цель урока: Научить студентов организовывать файлы проекта, связывать HTML с CSS и JS, создавать многостраничные сайты.

Ключевой момент: Понимание структуры проекта критически важно для дальнейшей работы. Без этого студенты будут теряться в файлах.

Практика: Обязательно создайте вместе простой многостраничный сайт из 2-3 страниц.

Зачем нужна структура?

Представьте шкаф, где все вещи свалены в кучу. Найти нужную футболку сложно! То же самое с файлами проекта.

Без структуры
page.html
style.css
main.js
about.html
image1.jpg
style2.css
contact.html
logo.png
...хаос!
Со структурой
index.html
about.html
contact.html
css/styles.css
js/app.js
images/logo.png
...порядок!
Стандартная структура веб-проекта

Вот как обычно организуют файлы профессионалы:

Структура папок 
my-website/                   - Папка проекта
  index.html                  - Главная страница
  about.html                  - Страница "О нас"
  contact.html                - Страница контактов

  css/                        - Папка для стилей
    styles.css                - Основные стили

  js/                         - Папка для скриптов
    app.js                    - JavaScript код

  images/                     - Папка для картинок
    logo.png                  - Логотип
    photo.jpg                 - Фотография
index.html - это всегда главная страница. Браузеры ищут этот файл по умолчанию, когда открывают папку.
Типы файлов в веб-проекте
Расширение Что это Для чего
.html HTML-файл Структура страницы (скелет)
.css CSS-файл Стили и оформление (красота)
.js JavaScript-файл Интерактивность (действия)
.png, .jpg, .gif Картинки Изображения на странице
.svg Векторная графика Иконки и логотипы
Как подключить CSS к HTML

Чтобы стили из CSS-файла работали, нужно "связать" его с HTML-страницей:

HTML 
<!DOCTYPE html>
<html>
<head>
  <title>Моя страница</title>

  <!-- Вот так подключаем CSS -->
  <link rel="stylesheet" href="css/styles.css">

</head>
<body>
  <h1>Привет!</h1>
</body>
</html>
rel="stylesheet"

Говорит браузеру, что это файл стилей

href="css/styles.css"

Путь к файлу: папка css, файл styles.css

Частая ошибка: Неправильный путь к файлу! Если CSS лежит в папке css, путь должен быть "css/styles.css", а не просто "styles.css".
Как подключить JavaScript к HTML

JavaScript подключается похожим образом, но в конце страницы:

HTML 
<!DOCTYPE html>
<html>
<head>
  <title>Моя страница</title>
  <link rel="stylesheet" href="css/styles.css">
</head>
<body>
  <h1>Привет!</h1>
  <button id="myBtn">Нажми меня</button>

  <!-- JavaScript в конце body -->
  <script src="js/app.js"></script>
</body>
</html>
Почему в конце? Чтобы сначала загрузилась вся страница, а потом JavaScript начал с ней работать.
Пути к файлам: понятное объяснение

Путь - это "адрес" файла. Как сказать браузеру, где найти нужный файл:

Путь Что значит Пример
styles.css Файл в той же папке HTML и CSS рядом
css/styles.css Файл в папке css Зайти в папку css, взять styles.css
../styles.css Файл в папке выше Выйти на уровень вверх
images/photo.jpg Картинка в папке images Зайти в images, взять photo.jpg
Аналогия с домом:

Представьте, что вы в комнате (HTML-файл). Чтобы взять вещь из шкафа (папки), нужно сказать: "Открой шкаф → возьми с полки → нужную вещь". Это и есть путь: шкаф/полка/вещь.

Создание многостраничного сайта

Большинство сайтов состоит из нескольких страниц. Как связать их между собой?

Структура сайта:

Папка проекта 
my-site/
  index.html     - Главная
  about.html     - О нас
  contact.html   - Контакты
  css/
    styles.css   - Один файл стилей для всех страниц!

Навигация между страницами:

HTML - Меню навигации 
<nav>
  <a href="index.html">Главная</a>
  <a href="about.html">О нас</a>
  <a href="contact.html">Контакты</a>
</nav>
Важно: Один CSS-файл может использоваться на всех страницах! Подключите его ко всем HTML-файлам, и стили будут одинаковыми.
Практика: создаём структуру проекта

Давайте создадим простой сайт из 2 страниц:

  1. Создайте папку проекта
    Назовите её "my-first-site"
  2. Создайте папку css внутри
    Правый клик в боковой панели → New Folder → css
  3. Создайте файлы:
    index.html, about.html, css/styles.css
  4. Попросите ИИ:
    "Создай главную страницу index.html с меню навигации на главную и about.html. Подключи css/styles.css"
  5. Попросите ИИ:
    "Создай страницу about.html с такой же навигацией и текстом 'Это страница О нас'"
  6. Попросите ИИ:
    "В css/styles.css добавь красивые стили для навигации"
Проверка: Откройте сайт через Live Server и убедитесь, что можете переходить между страницами, кликая на ссылки в меню.
Частые ошибки новичков
Ошибка: Неправильный путь к CSS
href="styles.css" - файл не найден!
href="css/styles.css" - правильно!
Ошибка: Пробелы в именах файлов
my page.html - могут быть проблемы
my-page.html - правильно!
Ошибка: Русские буквы в именах
страница.html - плохо
page.html - хорошо!
Ошибка: Разный регистр
Файл Styles.css, а в коде styles.css - на некоторых системах не найдёт!
Правило: Используйте только английские буквы, цифры, дефисы (-) и подчёркивания (_) в именах файлов. Всё в нижнем регистре.
