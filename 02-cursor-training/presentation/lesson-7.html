Cmd+K - быстрые команды

Редактируем код без открытия чата

Что такое Cmd+K?

Cmd+K (или Ctrl+K на Windows) — это быстрый способ изменить код прямо на месте, без открытия чата.

Выделяете код → Cmd+K → Говорите что сделать → Готово!
Как использовать
1
Выделите код

Мышкой или Shift+стрелки

2
Нажмите Cmd+K (Ctrl+K)

Появится поле ввода

3
Напишите команду

Например: "сделай красным"

4
Нажмите Enter

Код изменится прямо на месте!

Полезные команды для Cmd+K
"объясни что это делает"

ИИ объяснит выделенный код

"добавь комментарии"

Добавит пояснения к коду

"сделай это красивее"

Улучшит внешний вид

"измени цвет на синий"

Изменит цвет элемента

"добавь анимацию"

Добавит эффект анимации

"сделай больше"

Увеличит размер элемента

Практика

Создайте кнопку и улучшите её с помощью Cmd+K:

  1. Напишите: <button>Кнопка</button>
  2. Выделите весь код кнопки
  3. Нажмите Cmd+K → "сделай больше"
  4. Выделите снова → Cmd+K → "добавь тень"
  5. Выделите снова → Cmd+K → "измени цвет на зелёный"
Совет: Cmd+K быстрее чата для маленьких изменений!
