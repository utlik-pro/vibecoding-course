Основы HTML

Изучаем "скелет" веб-страницы

Заметка для преподавателя

Цель урока: Дать базовое понимание HTML. Студенты должны узнавать основные теги и понимать структуру документа.

Важно: Мы НЕ учим их писать HTML с нуля - это делает ИИ. Мы учим их ПОНИМАТЬ и ЧИТАТЬ код, чтобы они могли давать правильные инструкции ИИ.

Практика: Много примеров, много "попроси ИИ сделать..." вместо "напиши код..."

Что такое HTML?

HTML (HyperText Markup Language) — это язык разметки, который определяет структуру веб-страницы.

HTML - это скелет

Как скелет определяет форму тела, HTML определяет структуру страницы: где заголовок, где текст, где картинка.

ИИ знает HTML идеально

Вам не нужно запоминать все теги! Достаточно понимать основы, чтобы правильно общаться с ИИ.

Теги - строительные блоки HTML

HTML состоит из тегов. Тег - это специальная команда в угловых скобках:

Анатомия тега 
<название>содержимое</название>

Например:
<h1>Это заголовок</h1>
<p>Это параграф текста</p>
<button>Нажми меня</button>
<h1> - Открывающий тег (начало)
Это заголовок - Содержимое тега (то, что видит пользователь)
</h1> - Закрывающий тег (конец, со слешем /)
Аналогия: Теги как скобки в математике - нужен и открывающий, и закрывающий. Как (2 + 3), так и <p>текст</p>.
Структура HTML-документа

Каждая HTML-страница имеет стандартную структуру:

HTML 
<!DOCTYPE html>
<html>
<head>
  <title>Название страницы</title>
</head>
<body>
  Здесь всё, что видит пользователь
</body>
</html>
<!DOCTYPE html>

Говорит браузеру: "Это HTML5" (современная версия)

<html>

Корневой элемент - вся страница внутри него

<head>

"Голова" - невидимая часть: настройки, заголовок вкладки, подключение стилей

<body>

"Тело" - видимая часть: всё, что видит пользователь

Атрибуты - настройки тегов

Атрибуты добавляют дополнительную информацию к тегам:

HTML 
<!-- Ссылка с атрибутом href -->
<a href="https://google.com">Перейти в Google</a>

<!-- Картинка с атрибутами src и alt -->
<img src="photo.jpg" alt="Описание картинки">

<!-- Кнопка с атрибутом id -->
<button id="myButton">Нажми</button>

<!-- Элемент с атрибутом class -->
<div class="container">Контент</div>
href - адрес ссылки (куда перейти)
src - путь к файлу (картинка, скрипт)
alt - текст, если картинка не загрузилась
id - уникальный идентификатор элемента
class - класс для стилизации (о нём в уроке CSS)
Формы и ввод данных

Формы позволяют пользователям вводить данные:

HTML 
<form>
  <!-- Текстовое поле -->
  <input type="text" placeholder="Ваше имя">

  <!-- Поле для email -->
  <input type="email" placeholder="Email">

  <!-- Поле для пароля -->
  <input type="password" placeholder="Пароль">

  <!-- Большое текстовое поле -->
  <textarea placeholder="Ваше сообщение"></textarea>

  <!-- Кнопка отправки -->
  <button type="submit">Отправить</button>
</form>

Семантические теги (для понимания)

Современный HTML использует "говорящие" теги, которые объясняют назначение блока:

Тег Назначение
<header> Шапка страницы (логотип, меню)
<nav> Навигационное меню
<main> Основное содержимое страницы
<section> Раздел страницы
<article> Самостоятельная статья/пост
<footer> Подвал страницы (контакты, копирайт)
Зачем это знать? Когда вы видите такие теги в коде, вы сразу понимаете, что это за часть страницы. ИИ часто использует их - теперь вы будете понимать код лучше!
