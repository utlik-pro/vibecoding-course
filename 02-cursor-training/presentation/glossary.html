Глоссарий терминов

Все термины модуля в одном месте

Совет: Используйте Ctrl+F (Cmd+F на Mac), чтобы быстро найти нужный термин на странице.
А Б В Г Д И К Л М О П Р С Т Ф Х Ц Ш Э
А

Адаптивный дизайн (Responsive Design)

Подход к созданию сайтов, при котором страница автоматически подстраивается под размер экрана устройства (компьютер, планшет, телефон).

Аналогия: Как резиновая перчатка, которая подходит на руки разного размера.

Атрибут (Attribute)

Дополнительная информация, которую можно добавить к HTML-тегу. Например, href в ссылке или src в картинке.

Пример: <a href="page.html"> - здесь href это атрибут.

Автодополнение (Autocomplete)

Функция редактора, которая предлагает варианты завершения кода по мере набора. В Cursor - это Tab-автодополнение с помощью ИИ.

Б

Браузер (Browser)

Программа для просмотра веб-страниц. Примеры: Chrome, Safari, Firefox, Edge. Браузер "читает" HTML/CSS/JS код и показывает красивую страницу.

Аналогия: Браузер - это "переводчик", который превращает код в то, что вы видите на экране.

Блочная модель (Box Model)

Концепция CSS, где каждый элемент представляется как прямоугольник с содержимым, внутренним отступом (padding), границей (border) и внешним отступом (margin).

Breakpoint (Точка перелома)

Ширина экрана, при которой меняется расположение элементов на странице. Например, на экране уже 768px карточки выстраиваются в колонку вместо ряда.

В

Вайбкодинг (Vibecoding)

Новый подход к программированию, где вы описываете что хотите получить на обычном языке, а ИИ пишет код за вас. Фокус на результате, а не на синтаксисе.

Суть: "Объясни что хочешь - ИИ напишет код"

Viewport

Видимая область веб-страницы в браузере. На телефоне viewport узкий, на компьютере - широкий. Важен для адаптивного дизайна.

Веб-страница

Документ, который можно открыть в браузере. Обычно это HTML-файл, который может включать CSS для стилей и JavaScript для интерактивности.

Г

Градиент (Gradient)

Плавный переход от одного цвета к другому. Используется в CSS для красивых фонов.

Пример: background: linear-gradient(to right, blue, purple);

Горячие клавиши

Комбинации клавиш для быстрого выполнения действий. Например, Cmd+S для сохранения, Cmd+K для быстрых команд в Cursor.

Д

DevTools (Инструменты разработчика)

Встроенные инструменты браузера для изучения и отладки веб-страниц. Открываются через Cmd+Option+I (Mac) или Ctrl+Shift+I (Windows).

DOM (Document Object Model)

Структура HTML-документа в виде дерева элементов. Браузер создаёт DOM из вашего HTML-кода.

И

ИИ (Искусственный интеллект, AI)

Компьютерная программа, которая может понимать текст, отвечать на вопросы и генерировать код. В Cursor используется для помощи в программировании.

Идентификатор (ID)

Уникальное имя элемента на странице. В HTML: id="myElement", в CSS для выбора: #myElement.

Важно: ID должен быть уникальным на странице (только один элемент с таким id).

Интерфейс

То, что видит и с чем взаимодействует пользователь. Кнопки, меню, формы - всё это части интерфейса.

К

Класс (Class)

Имя, которое можно присвоить нескольким элементам для их стилизации. В HTML: class="card", в CSS: .card { ... }.

Отличие от ID: Один класс можно использовать для многих элементов.

Код

Текстовые инструкции для компьютера, написанные на языке программирования (HTML, CSS, JavaScript и др.).

Аналогия: Код - это рецепт для компьютера.

Composer

Мощный режим ИИ в Cursor, который может создавать и редактировать несколько файлов одновременно. Открывается через Cmd+I (Mac) или Ctrl+I (Windows).

Cursor

Редактор кода со встроенным ИИ-помощником. Основной инструмент для вайбкодинга в этом курсе.

Л

Live Server

Расширение для Cursor/VS Code, которое автоматически обновляет страницу в браузере при сохранении файла. Очень удобно для разработки!

М

Media Query

CSS-правило, которое применяет стили только при определённых условиях (например, при определённой ширине экрана).

Пример: @media (max-width: 768px) { ... }

Margin (Внешний отступ)

Расстояние снаружи элемента - между этим элементом и соседними. CSS-свойство: margin.

О

Ошибка (Error, Bug)

Проблема в коде, из-за которой программа работает неправильно. Это нормальная часть программирования! ИИ поможет найти и исправить.

П

Padding (Внутренний отступ)

Расстояние внутри элемента - между содержимым и границей. CSS-свойство: padding.

Аналогия: Padding - это "подушка" внутри коробки.

Путь к файлу (Path)

"Адрес" файла в структуре папок. Например: css/styles.css означает "файл styles.css в папке css".

Проект

Папка со всеми файлами вашей программы или сайта. В Cursor вы открываете папку проекта для работы.

Программирование

Процесс создания инструкций для компьютера. В вайбкодинге - это общение с ИИ для получения нужного кода.

Р

Расширение (Extension)

Дополнительная программа, которая добавляет функции в редактор. Например, Live Server - это расширение для Cursor.

Редактор кода

Программа для написания и редактирования кода. Cursor, VS Code, Sublime Text - примеры редакторов кода.

С

Селектор (Selector)

В CSS - указатель на элементы, к которым применяются стили. Примеры: p (все параграфы), .card (все с классом card), #logo (элемент с id logo).

Свойство CSS (Property)

Характеристика, которую можно изменить через CSS. Например: color, font-size, margin.

Синтаксис

Правила написания кода на определённом языке. В вайбкодинге за синтаксис отвечает ИИ!

Т

Тег (Tag)

Элемент HTML-разметки в угловых скобках. Пример: <p> - открывающий тег параграфа, </p> - закрывающий.

Tab-автодополнение

Функция Cursor, при которой ИИ предлагает продолжение кода, и вы принимаете его нажатием Tab.

Ф

Файл

Документ на компьютере, содержащий данные. В веб-разработке: HTML-файлы, CSS-файлы, картинки и т.д.

Flexbox

Способ расположения элементов в CSS. Включается через display: flex. Позволяет легко выстраивать элементы в ряд или колонку.

Форма (Form)

HTML-элемент для ввода данных пользователем. Содержит поля ввода, кнопки и т.д. Тег: <form>.

Х

HTML (HyperText Markup Language)

Язык разметки для создания структуры веб-страниц. Определяет, ЧТО находится на странице (заголовки, текст, картинки, ссылки).

Аналогия: HTML - это скелет страницы.

HEX-код

Способ записи цвета в шестнадцатеричном формате. Пример: #FF0000 = красный, #000000 = чёрный.

Hover

Состояние элемента, когда на него наведён курсор мыши. В CSS: button:hover { ... }.

Ц

CSS (Cascading Style Sheets)

Язык стилей для оформления веб-страниц. Определяет, КАК выглядит страница (цвета, размеры, расположение).

Аналогия: CSS - это одежда и макияж для HTML.

Цикл разработки

Повторяющийся процесс: написал код - сохранил - посмотрел результат - исправил - повторил. Основа работы программиста.

Ш

Шрифт (Font)

Внешний вид букв. В CSS настраивается через свойства: font-family (семейство), font-size (размер), font-weight (жирность).

Э

Элемент

Отдельная часть веб-страницы: кнопка, заголовок, картинка, форма. В HTML каждый элемент задаётся тегом.
