Cursor - ваш ИИ-помощник

Научитесь создавать сайты с помощью искусственного интеллекта.
Никакого предыдущего опыта не требуется!

Заметка для преподавателя

Общая продолжительность модуля: ~20 часов

Рекомендация: Этот модуль для полных новичков. Используйте простые аналогии, показывайте экран и делайте всё вместе со студентами. Не торопитесь - лучше меньше, но понятнее.

Ключевая цель: К концу модуля студенты смогут самостоятельно создавать красивые веб-страницы с помощью ИИ в Cursor.

Что вы освоите в этом модуле:
  • Поймёте что такое программирование и вайбкодинг
  • Установите и настроите Cursor
  • Научитесь общаться с ИИ для создания кода
  • Освоите Tab-автодополнение и Composer
  • Создадите свои первые веб-страницы
  • Поймёте основы HTML и CSS
  • Создадите полноценный сайт-портфолио
Проект модуля: Сайт-портфолио
Вы создадите красивый многостраничный сайт о себе с использованием ИИ. Реальный проект, который можно показать друзьям и работодателям!
