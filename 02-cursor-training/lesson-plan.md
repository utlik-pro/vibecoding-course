## Итоговый проект модуля

**Задание:** Создать полноценный Todo-приложение с использованием всех изученных возможностей Cursor:
- Использовать Composer для генерации структуры
- Применить .cursorrules для консистентного стиля
- Задействовать Chat для решения сложных задач
- Эффективно использовать Tab автодополнение
- Организовать код через Cmd+K команды

**Требования:**
- React + TypeScript
- Состояние в localStorage
- Современный UI (Tailwind CSS)
- Полностью функциональный CRUD
- Документированный код
- Тесты (с помощью ИИ)

## Критерии оценки

- Использование всех изученных функций Cursor
- Качество и читаемость кода
- Эффективность workflow
- Персонализация .cursorrules
- Функциональность приложения
