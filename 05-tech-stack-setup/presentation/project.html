Итоговый проект

Проект готов к вайбкодингу

Результат модуля:

  • Next.js проект настроен и задеплоен на Vercel
  • ИИ-инструменты (Cursor, Claude Code) сконфигурированы
  • Проект полностью готов к разработке с помощью ИИ
Требования к проекту

1. Проект и деплой

  • Next.js проект создан и запускается локально
  • npm run build проходит без ошибок
  • Vercel деплой работает

2. ИИ-инструменты

  • .cursorrules файл создан и настроен
  • CLAUDE.md с документацией проекта
  • prompts.md с шаблонами промптов
  • Cursor понимает контекст проекта
  • Claude Code работает с проектом

3. Интеграции (опционально)

  • Supabase клиент подключен
  • OpenAI API настроен
  • Health check endpoint работает
