Интеграция сервисов

Цели урока:

  • Подключить Supabase к проекту
  • Настроить OpenAI API
  • Создать health check endpoint
1. Supabase интеграция

Установка:

bash 
npm install @supabase/supabase-js

Создание клиента:

typescript 
// src/lib/supabase.ts
import { createClient } from '@supabase/supabase-js';

const supabaseUrl = process.env.NEXT_PUBLIC_SUPABASE_URL!;
const supabaseAnonKey = process.env.NEXT_PUBLIC_SUPABASE_ANON_KEY!;

export const supabase = createClient(supabaseUrl, supabaseAnonKey);

// Для серверных операций (с полными правами)
export const supabaseAdmin = createClient(
  supabaseUrl,
  process.env.SUPABASE_SERVICE_ROLE_KEY!
);
Важно: supabaseAdmin имеет полный доступ к БД. Используйте только на сервере!
2. OpenAI API

Установка:

bash 
npm install openai

Создание клиента:

typescript 
// src/lib/openai.ts
import OpenAI from 'openai';

export const openai = new OpenAI({
  apiKey: process.env.OPENAI_API_KEY,
});

Пример API Route:

typescript 
// src/app/api/chat/route.ts
import { openai } from '@/lib/openai';
import { NextResponse } from 'next/server';

export async function POST(request: Request) {
  try {
    const { message } = await request.json();

    const completion = await openai.chat.completions.create({
      model: 'gpt-4o-mini',
      messages: [{ role: 'user', content: message }],
    });

    return NextResponse.json({
      reply: completion.choices[0].message.content
    });
  } catch (error) {
    return NextResponse.json(
      { error: 'Failed to process request' },
      { status: 500 }
    );
  }
}
3. Health Check Endpoint

Создайте endpoint для проверки всех интеграций:

typescript 
// src/app/api/health/route.ts
import { supabase } from '@/lib/supabase';
import { openai } from '@/lib/openai';
import { NextResponse } from 'next/server';

export async function GET() {
  const checks = {
    supabase: false,
    openai: false,
    timestamp: new Date().toISOString(),
  };

  // Check Supabase
  try {
    const { error } = await supabase.from('_test').select().limit(1);
    // Даже если таблица не существует, соединение работает
    checks.supabase = !error || error.code === 'PGRST116';
  } catch {
    checks.supabase = false;
  }

  // Check OpenAI
  try {
    const models = await openai.models.list();
    checks.openai = models.data.length > 0;
  } catch {
    checks.openai = false;
  }

  const allHealthy = checks.supabase && checks.openai;

  return NextResponse.json(checks, {
    status: allHealthy ? 200 : 503
  });
}

Проверьте: http://localhost:3000/api/health

Отлично! Если видите {"supabase": true, "openai": true} — всё работает!
