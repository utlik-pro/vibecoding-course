Режим студента

Настройка проекта для вайбкодинга

Подготовка проекта к эффективной работе с ИИ-инструментами.
Cursor, Claude Code, промпты и интеграции.

Заметка для преподавателя

Общая продолжительность модуля: ~3 часа

Рекомендация: Этот модуль полностью практический. Показывайте экран и выполняйте команды вместе со студентами. Особое внимание уделите настройке ИИ-инструментов.

Ключевая цель: К концу модуля у каждого студента должен быть проект, полностью готовый к вайбкодингу с настроенными Cursor и Claude Code.

Что вы сделаете в этом модуле:
  • Создадите Next.js проект с TypeScript
  • Настроите .cursorrules для Cursor
  • Создадите CLAUDE.md для Claude Code
  • Научитесь писать эффективные промпты
  • Подключите Supabase и OpenAI
