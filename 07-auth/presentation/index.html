Аутентификация с Supabase

Добавляем систему входа и регистрации в приложение.
Email, пароли, Google, GitHub — всё просто!

Заметка для преподавателя

Общая продолжительность модуля: ~3-4 часа

Рекомендация: Аутентификация может показаться сложной темой, но с Supabase Auth это просто! Используйте аналогию с пропуском в здание. Делайте акцент на безопасности.

Ключевая цель: К концу модуля у каждого студента будет работающая система регистрации/входа с защитой страниц.

Что вы создадите в этом модуле:
  • Поймёте что такое аутентификация и зачем она нужна
  • Настроите Supabase Auth в существующем проекте
  • Создадите формы регистрации и входа
  • Добавите вход через Google и GitHub
  • Защитите страницы от неавторизованных пользователей
  • Научитесь работать с данными текущего пользователя
Почему Supabase Auth?
  • Всё в одном месте — база данных и аутентификация
  • Проще настроить, чем отдельные сервисы
  • Автоматическая интеграция с базой данных
  • Бесплатно для небольших проектов
