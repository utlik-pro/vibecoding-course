Программа курса
"Vibecoding с нуля"

Создание SaaS-приложений с помощью ИИ-инструментов

Продолжительность: 6-7 недель (42-49 дней)

Формат: Практический интенсив

Уровень: От новичка до продвинутого

Описание курса

Vibecoding — это революционный подход к программированию, который использует силу ИИ-инструментов для создания качественного кода быстро и эффективно. Вместо написания каждой строчки кода вручную, студенты научатся "кодить через вайб" - описывать желаемый результат, а ИИ поможет его реализовать.

Цели курса:

  • Освоить современный подход к разработке с использованием ИИ
  • Создать полноценное SaaS-приложение от идеи до деплоя
  • Изучить весь современный технологический стек веб-разработки
  • Автоматизировать рутинные задачи разработки
  • Подготовиться к работе программистом в эпоху ИИ

Технологический стек:

  • Frontend: Next.js, React, TypeScript, Tailwind CSS, shadcn/ui
  • Backend: Next.js API Routes, Drizzle ORM
  • Database: Supabase/PostgreSQL
  • Authentication: Clerk
  • Payments: Stripe
  • Deployment: Vercel
  • AI Tools: Cursor, Claude Code, OpenAI API

Итоговый результат:

Полноценное SaaS-приложение с лендингом, аутентификацией, базой данных, API, платежной системой и деплоем в production.

Система оценивания

Критерии оценки проектов:

  • Функциональность (40%) - Работает ли как задумано?
  • Качество кода (30%) - Читаемость, структура, best practices
  • UI/UX (20%) - Пользовательский опыт
  • Использование ИИ (10%) - Эффективность работы с ИИ-инструментами

Уровни оценки:

  • Отлично (90-100%) - Превосходит ожидания
  • Хорошо (80-89%) - Соответствует всем требованиям
  • Удовлетворительно (70-79%) - Основные требования выполнены
  • Нуждается в доработке (<70%) - Не все требования выполнены
