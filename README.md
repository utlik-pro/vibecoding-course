# Курс "Vibecoding с нуля"

## Что такое Vibecoding?

Vibecoding — это новый подход к программированию, который использует силу ИИ-инструментов для создания качественного кода быстро и эффективно. Вместо написания каждой строчки кода вручную, вы научитесь "кодить через вайб" - описывать желаемый результат, а ИИ поможет его реализовать.

## Цель курса

По окончании курса вы сможете:
- Создать полноценное SaaS-приложение от идеи до деплоя
- Использовать современные ИИ-инструменты для разработки
- Работать с полным технологическим стеком
- Интегрировать аутентификацию, базы данных и платежные системы
- Деплоить приложения в production

## Результат

В итоге у вас получится полноценное SaaS-приложение с:
- Лендинг-страницей
- Системой аутентификации
- База данных
- API
- Платежной системой
- Деплоем на Vercel

## Требования

- Базовые знания HTML/CSS
- Понимание основ программирования
- Желание изучать новые технологии
- Готовность экспериментировать с ИИ

## Структура курса

1. **Введение** - Основы vibecoding и инструменты
2. **Cursor тренинг** - Глубокое погружение в Cursor
3. **Claude Code** - Альтернативный ИИ-помощник
4. **Обзор технологий** - Современный веб-стек
5. **Настройка стека** - Практическая настройка инструментов
6. **Бекенд** - Серверная часть приложения
7. **Аутентификация** - Регистрация и вход пользователей
8. **База данных** - Хранение и управление данными
9. **Фронтенд** - Пользовательский интерфейс
10. **API и платежи** - Интеграции и монетизация
11. **Интеграция** - Связываем всё вместе
12. **Финальная проверка** - Тестирование и презентация проекта

## Технологический стек

### Основные технологии
- **Frontend**: Next.js, React, TypeScript, Tailwind CSS
- **Backend**: Next.js API Routes, Drizzle ORM
- **Database**: Supabase/PostgreSQL
- **Authentication**: Clerk
- **Payments**: Stripe
- **Deployment**: Vercel
- **AI Tools**: Cursor, Claude Code, OpenAI API

### Дополнительные инструменты
- GitHub (версионирование)
- shadcn/ui (компоненты)
- Environment variables
- Webhooks

## Как проходить курс

1. Каждый модуль содержит теоретическую и практическую части
2. Выполняйте все практические задания
3. Создавайте коммиты в Git после каждого урока
4. Используйте ИИ-помощников для решения проблем
5. Экспериментируйте и не бойтесь ошибок
