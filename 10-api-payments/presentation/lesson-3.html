GitHub - облачное хранилище кода

25 минут

Регистрация на GitHub

  1. Переходим на github.com
  2. Нажимаем "Sign up"
  3. Вводим email, пароль, username
  4. Подтверждаем email
  5. Выбираем Free план
Важно: выбор username

Это ваше профессиональное имя! Работодатели смотрят GitHub.

Хорошо: ivan-petrov, maria-dev, alex-frontend

Плохо: xXx_killer_xXx, qwerty123, test12345

Создание репозитория

  1. Нажимаем + в правом верхнем углу
  2. "New repository"
  3. Заполняем:
    • Repository name: data-aggregator
    • Description: Агрегатор данных - курс Vibecoding
    • Public (чтобы показать работодателям)
    • НЕ ставим галочки (README, .gitignore) - у нас уже есть!
  4. Create repository

Подключаем GitHub к Cursor

Промпт для Cursor
Подключи мой локальный Git репозиторий к GitHub.
URL репозитория: https://github.com/username/data-aggregator.git
Покажи шаги и выполни команды.

Cursor выполнит:

git remote add origin https://github.com/username/data-aggregator.git
git branch -M main
git push -u origin main

Первый Push

Push = отправить commits на GitHub

Способ 1: Через интерфейс

Source Control → три точки ... → "Push"

Способ 2: Горячие клавиши

Cmd+Shift+P → "Git: Push" → Enter

Проверяем:

  1. Открываем github.com/username/data-aggregator
  2. Обновляем страницу
  3. Видим все наши файлы!
