Stripe - настройка и основы

25 минут

Что такое Stripe

Stripe = платёжная система для разработчиков

  • Принимает карты со всего мира
  • Автоматически конвертирует валюту
  • Обрабатывает подписки
  • Безопасно хранит данные карт
Почему Stripe?

Самая популярная в мире, отличная документация, работает с Next.js из коробки.

Регистрация и Test Mode

  1. Переходим на stripe.com
  2. "Start now" → Create account
  3. Подтверждаем email
  4. ВАЖНО: Остаёмся в Test mode!
TEST MODE

Карты НЕ списывают реальные деньги
Тестовая карта: 4242 4242 4242 4242
Идеально для обучения

LIVE MODE

Настоящие деньги!
Нужна верификация бизнеса
НЕ включаем пока не готовы!

Получение API ключей

Dashboard → Developers → API keys

Publishable key: pk_test_51ABC...
(можно показывать в браузере)

Secret key: sk_test_51XYZ...
(ТОЛЬКО на сервере!)

Добавляем в .env.local:

# Stripe (Test Mode)
NEXT_PUBLIC_STRIPE_PUBLISHABLE_KEY=pk_test_ваш_ключ
STRIPE_SECRET_KEY=sk_test_ваш_ключ

Установка Stripe

npm install stripe @stripe/stripe-js
Промпт для настройки
Настрой Stripe в Next.js проекте:
1. Создай lib/stripe.ts — серверный клиент
2. Создай lib/stripe-client.ts — клиентский клиент
3. Используй environment variables
