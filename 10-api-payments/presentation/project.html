Итоговый проект: Data Aggregator

Применяем все знания

Описание проекта

Создайте веб-приложение, которое агрегирует данные из внешних API и предлагает Premium подписку.

FREE версия

  • Погода для 1 города
  • Курсы 3 валют
  • Обновление раз в час

PREMIUM $9.99/мес

  • Погода для любого города
  • Все доступные валюты
  • Обновление в реальном времени
  • Без рекламы

Требования к проекту

Git и GitHub (20 баллов)

Внешние API (30 баллов)

Stripe интеграция (30 баллов)

UI/UX (20 баллов)

Промпт для создания

Промпт для Cursor
Создай приложение Data Aggregator с Next.js:

1. Главная страница:
   - WeatherWidget (погода)
   - ExchangeWidget (курсы валют)
   - Баннер "Попробуйте Premium"

2. Страница /pricing:
   - Сравнение Free и Premium
   - Кнопка PurchaseButton

3. API Routes:
   - /api/weather?city=Moscow
   - /api/exchange?base=USD
   - /api/stripe/create-checkout-session
   - /api/stripe/webhook

Стили: Tailwind CSS, тёмная тема.
