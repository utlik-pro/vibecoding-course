Интеграция API в Next.js

35 минут

Получение API ключей

OpenWeatherMap (погода):

  1. Регистрация на openweathermap.org
  2. Подтверждаем email
  3. API Keys → Generate
  4. Копируем ключ
Важно!

Ключ активируется через 10-30 минут после создания!

ExchangeRate-API (курсы валют):

  1. Регистрация на exchangerate-api.com
  2. Dashboard → API Key
  3. Копируем ключ

Добавляем в .env.local:

# API ключи (НЕ публикуйте!)
OPENWEATHER_API_KEY=ваш_ключ_погоды
EXCHANGE_API_KEY=ваш_ключ_валют

API Route для погоды

Промпт для Cursor
Создай API route /api/weather в Next.js App Router:

1. Файл: app/api/weather/route.ts
2. Принимает query параметр city (город)
3. Делает запрос к OpenWeatherMap API
4. API ключ из process.env.OPENWEATHER_API_KEY
5. Возвращает: city, temp, description, humidity
6. Обрабатывает ошибки (город не найден, API недоступен)

Тестируем:

http://localhost:3000/api/weather?city=Moscow
http://localhost:3000/api/weather?city=Минск

API Route для курсов валют

Промпт для Cursor
Создай API route /api/exchange в Next.js App Router:

1. Файл: app/api/exchange/route.ts
2. Принимает query параметр base (по умолчанию USD)
3. Делает запрос к ExchangeRate-API
4. Возвращает курсы к RUB, EUR, BYN, UAH
5. Кеширует на 1 час (revalidate: 3600)

Виджеты для отображения

Промпт для WeatherWidget
Создай React компонент WeatherWidget:
1. Props: city (string)
2. Запрос к /api/weather?city={city}
3. Loading spinner во время загрузки
4. Показывает: город, температура, описание
5. Обработка ошибок
6. Стили: Tailwind CSS, красивая карточка
