Ветки и работа с изменениями

25 минут

Что такое ветки?

Аналогия: Параллельные вселенные

main ─────────────────────────────────────> (стабильная версия)
     \                                    /
      └── feature ──────────────────────┘
          (эксперимент)

Если эксперимент удался → merge (объединяем)
Если нет → удаляем ветку, main не пострадал
Зачем это вайбкодеру?

AI иногда делает странные вещи. Создаёте ветку → даёте AI экспериментировать → Если всё ок - merge, если нет - удаляете ветку.

Создание ветки

Способ 1: Через интерфейс Cursor

  1. Внизу слева видим название ветки (main)
  2. Кликаем на него
  3. "Create new branch..."
  4. Вводим имя: feature/add-weather

Способ 2: Через AI

Промпт
Создай новую Git ветку с названием feature/add-weather-widget
и переключись на неё

Правила именования веток:

ХорошоПлохо
feature/add-loginnew-stuff
fix/payment-bugtest
update/readmeaaa

Merge - объединение веток

Когда эксперимент удался и код работает:

  1. Переключаемся на main
  2. Source Control → три точки → "Branch" → "Merge Branch..."
  3. Выбираем нашу ветку (feature/add-weather)
  4. Push
Промпт для AI
Слей ветку feature/add-weather в main:
1. Переключись на main
2. Сделай merge feature/add-weather
3. Push изменения
