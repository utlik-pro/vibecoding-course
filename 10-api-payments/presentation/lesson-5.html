Как выглядит запрос и ответ

Запрос:

GET https://api.openweathermap.org/data/2.5/weather?q=Moscow&appid=YOUR_KEY

Ответ (JSON):

{
  "name": "Moscow",
  "main": {
    "temp": 5.2,
    "humidity": 80
  },
  "weather": [{
    "description": "облачно"
  }]
}

API ключи - безопасность

НИКОГДА так!
const API_KEY = "abc123secretkey"  // В коде — ОПАСНО!
ВСЕГДА так!
// .env.local (этот файл НЕ попадает на GitHub)
WEATHER_API_KEY=abc123secretkey

// В коде:
const API_KEY = process.env.WEATHER_API_KEY

Проверьте себя
