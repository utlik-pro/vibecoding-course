Checkout Session и приём платежей

30 минут

Как работает Stripe Checkout

Пользователь    →   Ваш сервер   →   Stripe
     │                   │              │
     │── "Купить" ──────>│              │
     │                   │── Сессия ───>│
     │                   │<── URL ──────│
     │<── Редирект ──────│              │
     │                                  │
     │─────── Вводит карту ────────────>│
     │                                  │
     │<────── Успех! ───────────────────│

Создание продукта в Stripe

  1. Dashboard → Products → Add product
  2. Name: "Premium подписка"
  3. Price: $9.99 / Monthly
  4. Save product
  5. Копируем Price ID: price_1ABC...

Добавляем в .env.local:

STRIPE_PRICE_ID=price_ваш_id
NEXT_PUBLIC_APP_URL=http://localhost:3000

API Route для создания сессии

Промпт для Cursor
Создай API route /api/stripe/create-checkout-session:

1. Метод: POST
2. Принимает: { priceId: string }
3. Создаёт Stripe Checkout Session:
   - mode: 'subscription'
   - success_url: /success
   - cancel_url: /pricing
4. Возвращает: { url: session.url }

Кнопка покупки

Промпт для PurchaseButton
Создай компонент PurchaseButton:
1. При клике вызывает /api/stripe/create-checkout-session
2. Редиректит на Stripe Checkout
3. Показывает loading
4. Текст: "Оформить Premium — $9.99/мес"

Тестирование платежа

  1. Нажимаем кнопку покупки
  2. Попадаем на Stripe Checkout
  3. Вводим тестовую карту
  4. Попадаем на /success
Тестовые данные карты

Номер: 4242 4242 4242 4242
Срок: любая дата в будущем
CVV: любые 3 цифры

Проверяем в Stripe Dashboard → Payments
