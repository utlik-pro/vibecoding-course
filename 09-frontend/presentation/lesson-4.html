Анимации с Framer Motion

Добавляем жизнь интерфейсу через плавные анимации

Заметка для преподавателя

Framer Motion — стандарт анимаций в React. Простой API, мощные возможности.

Совет: Показывайте "до" и "после" — студенты должны увидеть разницу.

Промпт 1: Установка и базовые анимации
Промпт для Cursor 
Настрой Framer Motion в Next.js проекте.

Создай компонент AnimatedSection:
- Появление при скролле (fade + slide up)
- Использует Intersection Observer
- Анимируется только первый раз
- Настраиваемая задержка и длительность
Intersection Observer: Отслеживает когда элемент появляется в viewport.
Промпт 2: Анимированные карточки
Промпт для Cursor 
Создай компонент AnimatedCard:
- При появлении: fade in + scale from 0.95
- При hover: небольшое увеличение + тень
- При клике: быстрый scale эффект
- Stagger эффект для списка карточек (каждая следующая с задержкой)
Промпт 3: Переходы между страницами
Промпт для Cursor 
Добавь анимации переходов между страницами:
- Fade out текущей страницы
- Fade in новой страницы
- Опционально: slide эффект
- Используй framer-motion AnimatePresence
Промпт 4: Микро-анимации

Детали, которые делают UI живым:

Промпт для Cursor 
Создай набор микро-анимаций:

1. Кнопка с pulse эффектом при hover
2. Иконка загрузки (spinning)
3. Checkbox с checkmark анимацией
4. Success/Error shake анимация для форм
5. Typing эффект для текста
6. Counter анимация (числа увеличиваются)
Промпт 5: Анимации для Hero

Первое впечатление — самое важное:

Промпт для Cursor 
Создай анимированный Hero блок:
- Заголовок появляется по буквам (или словам)
- Подзаголовок fade in с задержкой
- Кнопки slide up
- Фоновый gradient анимируется
- Floating элементы (иконки летают)
Результат: Wow-эффект при первом открытии сайта!
