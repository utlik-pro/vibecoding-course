Dashboard

Создаём админ-панель с таблицами, графиками и статистикой

Заметка для преподавателя

Dashboard — самая функциональная часть SaaS. Здесь живут данные пользователя.

Библиотека графиков: Recharts — простая и хорошо работает с React.

Промпт 1: Layout с Sidebar
Промпт для Cursor 
Создай Dashboard layout:

Sidebar (слева, 250px):
- Логотип вверху
- Навигация с иконками:
  - Dashboard (главная)
  - Задачи
  - Проекты
  - Команда
  - Аналитика
  - Настройки
- Разделитель
- Профиль пользователя внизу
- Кнопка collapse (свернуть в иконки)

Header (сверху):
- Breadcrumbs
- Search
- Notifications (колокольчик с badge)
- User dropdown

Main content area справа.
Промпт 2: Stats Cards
Промпт для Cursor 
Создай 4 Stats карточки для dashboard:

1. Всего задач - число, иконка clipboard
2. Выполнено - число + процент от общего, зелёная иконка check
3. В работе - число, синяя иконка clock
4. Просрочено - число, красная иконка alert (если есть)

Каждая карточка:
- Иконка с цветным фоном
- Число большим шрифтом
- Label мелким
- Изменение за неделю (+12% или -5%)
- Hover эффект
Промпт 3: Таблица задач
Промпт для Cursor 
Создай DataTable для задач:

Колонки:
- Checkbox (множественный выбор)
- Название задачи
- Статус (badge: new, in progress, done)
- Приоритет (high/medium/low с цветом)
- Исполнитель (avatar + имя)
- Дедлайн
- Actions (edit, delete)

Функции:
- Сортировка по любой колонке
- Поиск по названию
- Фильтр по статусу
- Pagination (10/25/50 на страницу)
- Bulk actions для выбранных
Совет: Используй @tanstack/react-table для продвинутых таблиц.
Промпт 4: Графики
Промпт для Cursor 
Создай секцию с графиками (используй Recharts):

1. Line Chart - "Выполненные задачи за месяц"
   - Ось X: дни месяца
   - Ось Y: количество
   - Tooltip с деталями

2. Bar Chart - "Задачи по проектам"
   - Горизонтальные бары
   - 5 проектов
   - Цветовая кодировка

3. Pie/Donut Chart - "Распределение по статусам"
   - Done, In Progress, Todo
   - Легенда справа
   - Проценты

Добавь переключатель периода: Неделя / Месяц / Год
Промпт 5: Activity Feed
Промпт для Cursor 
Создай компонент Activity Feed:

Список последних действий:
- Иконка действия (создал, завершил, комментировал)
- Текст: "Анна завершила задачу 'Дизайн главной'"
- Время: "2 часа назад"
- Аватар пользователя

Показывать последние 10.
Кнопка "Показать все".
Real-time обновление (визуально).
Промпт 6: Quick Actions
Промпт для Cursor 
Создай панель быстрых действий:

Кнопки:
- + Новая задача (открывает modal)
- + Новый проект
- Пригласить в команду
- Экспорт отчёта

При hover показывать tooltip с описанием.
Keyboard shortcuts (Cmd+N для новой задачи).
