Заметка для преподавателя

Общая продолжительность модуля: 7 дней

Формат: Вайбкодинг — студенты копируют промпты в Cursor/Claude и получают готовый код. Минимум ручного написания.

Ключевая цель: Создать Landing Page + Dashboard используя AI-промпты.

Что вы создадите в этом модуле:
  • Landing Page: Hero, Features, Pricing, FAQ, Footer
  • Dashboard: Sidebar, Stats, Таблицы, Графики
  • Интерактивные компоненты: Modal, Tabs, Accordion
  • Формы с валидацией (React Hook Form + Zod)
  • Анимации (Framer Motion)
  • Dark Mode и Responsive
Проект модуля: SaaS приложение TaskFlow
Полноценный фронтенд для таск-менеджера: красивый лендинг для привлечения клиентов + функциональная админ-панель.
