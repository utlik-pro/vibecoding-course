Landing Page

Создаём полноценный лендинг для SaaS через AI промпты

Заметка для преподавателя

Это главный урок! За 90 минут создаём полный лендинг из 7 секций.

Совет: Можно использовать v0.dev для быстрой генерации отдельных секций.

Промпт 1: Hero секция

Первое, что видит пользователь:

Промпт для Cursor 
Создай Hero секцию для SaaS лендинга TaskFlow:

Контент:
- Заголовок: "Управляй задачами с помощью AI"
- Подзаголовок: "TaskFlow автоматизирует рутину и помогает фокусироваться на важном"
- Кнопки: "Начать бесплатно" (primary), "Смотреть демо" (secondary)
- Справа: скриншот/mockup приложения или анимированная иллюстрация

Дизайн:
- Gradient фон (синий -> фиолетовый)
- Анимации при загрузке
- Responsive (на мобильном картинка под текстом)
Промпт 2: Features секция
Промпт для Cursor 
Создай Features секцию с 6 фичами:

1. AI Приоритизация - "AI автоматически определяет важность задач"
2. Умные напоминания - "Напоминания в нужное время"
3. Командная работа - "Совместная работа над проектами"
4. Интеграции - "Подключение к 50+ сервисам"
5. Аналитика - "Отчёты по продуктивности"
6. Мобильное приложение - "Работайте откуда угодно"

Дизайн:
- Сетка 3x2 на десктопе
- Каждая карточка с иконкой (Lucide)
- Hover эффект
- Анимация появления при скролле
Промпт 3: Pricing секция
Промпт для Cursor 
Создай Pricing секцию с 3 тарифами:

Free:
- Цена: 0₽
- До 10 задач
- 1 проект
- Базовые фичи

Pro:
- Цена: 990₽/мес (популярный - выделить!)
- Безлимит задач
- 10 проектов
- AI функции
- Приоритетная поддержка

Team:
- Цена: 2990₽/мес
- Всё из Pro
- До 10 участников
- Admin панель
- API доступ

Дизайн:
- Центральный (Pro) выше и с бейджем "Популярный"
- Toggle: месяц/год (скидка 20% за год)
- Кнопка CTA на каждом
Промпт 4: Testimonials секция
Промпт для Cursor 
Создай секцию отзывов:

5 отзывов с:
- Аватар
- Имя и должность
- Компания
- Текст отзыва
- Рейтинг (звёзды)

Примеры:
1. "TaskFlow сэкономил нам 10 часов в неделю" - Анна, CEO StartupX
2. "Лучший инструмент для управления задачами" - Михаил, PM TechCorp
...

Дизайн:
- Карусель или masonry grid
- Плавная анимация
- Кавычки как декор
Промпт 5: FAQ секция
Промпт для Cursor 
Создай FAQ с аккордеоном:

8 вопросов:
1. Как начать использовать TaskFlow?
2. Есть ли бесплатный период?
3. Как работает AI приоритизация?
4. Можно ли импортировать задачи?
5. Как добавить команду?
6. Какие интеграции доступны?
7. Как отменить подписку?
8. Есть ли мобильное приложение?

Придумай релевантные ответы для каждого.
