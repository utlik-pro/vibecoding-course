Интерактивные компоненты

Создаём переиспользуемые UI компоненты через AI промпты

Заметка для преподавателя

Цель урока: Создать 5 базовых компонентов, которые используются в любом SaaS.

Совет: После каждого промпта проверяйте результат в браузере вместе со студентами.

Промпт 1: Аккордеон (FAQ)

Идеально для: FAQ секции, списка вопросов, скрытого контента.

Промпт для Cursor 
Создай компонент Accordion для FAQ секции:
- Принимает массив {question, answer}
- Плавная анимация раскрытия (Framer Motion)
- Иконка + или стрелка поворачивается
- Только один item может быть открыт
- Tailwind стили, тёмная тема поддержка
Зачем Framer Motion: Плавные анимации без написания CSS keyframes.
Промпт 2: Табы

Идеально для: Переключение между разделами, фичи продукта.

Промпт для Cursor 
Создай компонент Tabs:
- Принимает массив {label, content}
- Анимированный индикатор под активным табом
- Плавная смена контента (fade)
- Поддержка клавиатурной навигации
- Variant: "underline" и "pills"
Промпт 3: Модальное окно

Идеально для: Подтверждения, формы, детали.

Промпт для Cursor 
Создай переиспользуемый Modal компонент:
- Props: isOpen, onClose, title, children
- Затемнение фона с blur
- Закрытие по Escape и клику вне окна
- Анимация появления (scale + fade)
- Кнопка закрытия в углу
- Используй Radix UI Dialog как основу
Промпт 4: Tooltip

Идеально для: Подсказки, объяснения иконок.

Промпт для Cursor 
Создай Tooltip компонент:
- Появляется при hover
- Позиции: top, bottom, left, right
- Задержка появления 200ms
- Стрелочка указывает на элемент
- Тёмный фон, белый текст
Промпт 5: Dropdown меню

Идеально для: Меню действий, выбор опций.

Промпт для Cursor 
Создай Dropdown компонент:
- Trigger кнопка с иконкой шеврона
- Список опций с иконками
- Разделители между группами
- Выделение при hover
- Закрытие при клике вне
- Поддержка клавиатуры (стрелки, Enter, Escape)
Практическое задание

Создайте демо-страницу всех компонентов:

Промпт для Cursor 
Создай страницу /components-demo где показаны все созданные компоненты:
- Accordion с 5 вопросами о TaskFlow
- Tabs с 3 вкладками (Фичи, Цены, Отзывы)
- Кнопка открытия Modal с формой
- Несколько элементов с Tooltip

Красивый layout с заголовками секций.
