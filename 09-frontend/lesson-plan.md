# Модуль 9: Фронтенд и интерактивные компоненты

## Обзор модуля

**Продолжительность:** 7 дней (расширенная версия с фокусом на интерактив)

**Цель:** Создать красивый, интерактивный фронтенд с множеством практических занятий. Больше времени на работу с компонентами, анимациями, формами и интерактивными элементами.

**Изменения:** Этот модуль расширен с 5 до 7 дней, чтобы дать студентам больше времени на практику с фронтендом и создание интерактивных элементов.

---

## Урок 9.1: Основы Next.js роутинга и лейауты

**Продолжительность:** 1 день

### Цели:
- Понять App Router в Next.js 14
- Создать структуру страниц
- Настроить layouts и metadata
- Научиться работать с Link и навигацией

### Содержание:

#### Структура App Router
```
app/
├── layout.tsx          # Корневой layout
├── page.tsx            # Главная страница
├── about/
│   └── page.tsx        # /about
├── blog/
│   ├── page.tsx        # /blog
│   ├── [slug]/
│   │   └── page.tsx    # /blog/[slug]
│   └── layout.tsx      # Layout для блога
└── dashboard/
    ├── layout.tsx      # Layout для dashboard
    ├── page.tsx        # /dashboard
    └── settings/
        └── page.tsx    # /dashboard/settings
```

#### Практика:
- Создать 5-6 основных страниц
- Настроить общий layout с Header и Footer
- Добавить динамические роуты для блога
- Настроить metadata для SEO

## Урок 9.4: Анимации и переходы

**Продолжительность:** 1 день

### Цели:
- Освоить Framer Motion для анимаций
- Создать плавные переходы между страницами
- Добавить микроинтеракции
- Использовать Tailwind CSS transitions

### Практические задания:

#### Анимации с Framer Motion:
1. **Fade in** - появление элементов
2. **Slide in** - въезд элементов сбоку
3. **Scale** - увеличение/уменьшение
4. **Stagger** - последовательная анимация списков
5. **Hover effects** - эффекты при наведении

#### Проекты:
- Анимированный Hero блок
- Карточки с hover эффектами
- Анимированная навигация
- Scrolling triggered анимации
- Page transitions

**Пример кода:**
```typescript
'use client'
import { motion } from 'framer-motion'

export default function AnimatedCard() {
  return (
    <motion.div
      initial={{ opacity: 0, y: 20 }}
      animate={{ opacity: 1, y: 0 }}
      whileHover={{ scale: 1.05 }}
      transition={{ duration: 0.3 }}
      className="p-6 bg-white rounded-lg shadow-lg"
    >
      <h3>Анимированная карточка</h3>
      <p>Наведи на меня!</p>
    </motion.div>
  )
}
```
