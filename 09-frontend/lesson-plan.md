## Урок 9.2: Интерактивные компоненты (часть 1)

**Продолжительность:** 1 день

### Цель:
Создать базовые интерактивные компоненты через AI.

### Промпты для компонентов:

#### 1. Аккордеон (FAQ)

```
Создай компонент Accordion для FAQ секции:
- Принимает массив {question, answer}
- Плавная анимация раскрытия (Framer Motion)
- Иконка + или стрелка поворачивается
- Только один item может быть открыт
- Tailwind стили, тёмная тема поддержка
```

#### 2. Табы

```
Создай компонент Tabs:
- Принимает массив {label, content}
- Анимированный индикатор под активным табом
- Плавная смена контента (fade)
- Поддержка клавиатурной навигации
- Variant: "underline" и "pills"
```

#### 3. Модальное окно

```
Создай переиспользуемый Modal компонент:
- Props: isOpen, onClose, title, children
- Затемнение фона с blur
- Закрытие по Escape и клику вне окна
- Анимация появления (scale + fade)
- Кнопка закрытия в углу
- Используй Radix UI Dialog как основу
```

#### 4. Tooltip

```
Создай Tooltip компонент:
- Появляется при hover
- Позиции: top, bottom, left, right
- Задержка появления 200ms
- Стрелочка указывает на элемент
- Тёмный фон, белый текст
```

#### 5. Dropdown меню

```
Создай Dropdown компонент:
- Trigger кнопка с иконкой шеврона
- Список опций с иконками
- Разделители между группами
- Выделение при hover
- Закрытие при клике вне
- Поддержка клавиатуры (стрелки, Enter, Escape)
```

### Практика:
Создай страницу `/components-demo` где показаны все компоненты.

### Чеклист:
- [ ] Accordion работает
- [ ] Tabs с анимацией
- [ ] Modal открывается/закрывается

## Урок 9.4: Анимации

**Продолжительность:** 1 день

### Цель:
Добавить анимации через Framer Motion с помощью AI.

### Промпты для анимаций:

#### 1. Установка и базовые анимации

```
Настрой Framer Motion в Next.js проекте.

Создай компонент AnimatedSection:
- Появление при скролле (fade + slide up)
- Использует Intersection Observer
- Анимируется только первый раз
- Настраиваемая задержка и длительность
```

#### 2. Анимированные карточки

```
Создай компонент AnimatedCard:
- При появлении: fade in + scale from 0.95
- При hover: небольшое увеличение + тень
- При клике: быстрый scale эффект
- Stagger эффект для списка карточек (каждая следующая с задержкой)
```

#### 3. Page Transitions

```
Добавь анимации переходов между страницами:
- Fade out текущей страницы
- Fade in новой страницы
- Опционально: slide эффект
- Используй framer-motion AnimatePresence
```

#### 4. Микро-анимации

```
Создай набор микро-анимаций:

1. Кнопка с pulse эффектом при hover
2. Иконка загрузки (spinning)
3. Checkbox с checkmark анимацией
4. Success/Error shake анимация для форм
5. Typing эффект для текста
6. Counter анимация (числа увеличиваются)
```

#### 5. Hero анимации

```
Создай анимированный Hero блок:
- Заголовок появляется по буквам (или словам)
- Подзаголовок fade in с задержкой
- Кнопки slide up
- Фоновый gradient анимируется
- Floating элементы (иконки летают)
```

### Практические примеры:

```
// Пример промпта для конкретной анимации
Сделай анимацию для списка фич:
- Каждая фича появляется с задержкой 0.1s
- Slide in слева
- Fade in одновременно
- При hover карточка приподнимается
```

### Чеклист:
- [ ] Framer Motion установлен
- [ ] AnimatedSection работает
- [ ] Карточки анимируются
- [ ] Page transitions есть

